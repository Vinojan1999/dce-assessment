-   Used [REQRES](https://reqres.in/) APIs.
-   User Registration UI, Using CREATE endpoint.
-   Update user UI, using UPDATE endpoint.
-   Display registered user UI, using GET endpoint.
-   Delete user function, using DELETE endpoint.
